# Portfolio — Sara Mosquera

Personal portfolio website built with React, showcasing my projects and skills as a junior fullstack developer.

## Live Site

https://portfolio-7auls8bu2-shadowk4ts-projects.vercel.app

## Built With

- React
- React Router
- CSS
- Google Fonts (Space Grotesk + DM Sans)

## Pages

- **Home** — introduction and links to my work
- **Projects** — cards for each project with stack and GitHub links
- **About** — background, bio and skills
- **Contact** — email, GitHub and LinkedIn

## Projects Featured

- **SquirrelStd** — fullstack task management web app for a game development studio (Next.js, Node.js, Express, SQL)
- **ParkSpot** — GPS-based Android parking app (Kotlin, Jetpack Compose, SQLite)
- **Weather App** — React app consuming a public weather API

## Run Locally

```bash
git clone https://github.com/ShadowK4t/Portfolio.git
cd Portfolio
npm install
npm start
```
